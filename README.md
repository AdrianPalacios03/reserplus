# react-native-reserplus

Library for using C++ Turbo Modules in Reserhub's saas-app

## Installation

```sh
yarn add github:AdrianPalacios03/reserplus#main;  
```

## Usage


```js
import { multiply } from 'react-native-reserplus';

// ...

const result = multiply(3, 7);
```
